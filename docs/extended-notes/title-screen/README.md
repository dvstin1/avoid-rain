# Title Screen Design Concepts

These mockups are designed to help visualize the "Scriptorium Noir" aesthetic for the **Avoid Rain** title screen. You can use these as blueprints for creating final PNG assets.

## Files
- **[overall-mockup.html](./overall-mockup.html)**: The full vision. Multiple layers (Parchement, Silhouettes, Toxic Rain, Typography) working together.
- **[part-background-ink.html](./part-background-ink.html)**: Ideas for the base layer. Uses SVG filters for noise/texture and animated ink spills.
- **[part-typographic-bloom.html](./part-typographic-bloom.html)**: Focusing on the high-contrast italicized title and its glowing "bloom" effect.
- **[part-character-silhouette.html](./part-character-silhouette.html)**: A concept for the silhouetted hooded figure holding the protective "Avoid Rain" umbrella.

## Suggested PNG Layers
1. **bg_layer.png**: Distorted parchment with static ink spills.
2. **mid_layer.png**: The silhouetted reader figure (transparent background).
3. **rain_layer.png**: A transparent sheet of vertical toxic green lines.
4. **ui_layer.png**: The high-contrast logo and menu text.
